# 🚀 Get Started

Welcome to the **Cardano Ambassador Tool**!

Apply for membership and participate in governance decisions that shape the Cardano Ambassador program.

## Quick Start Guide

### 1. 👤 Becoming a Member

**Start here!** Navigate to [Become an Ambassador](/sign-up) to apply for membership.

#### What You Need

- 💳 **Cardano Wallet:** Eternl, Nami, Lace, or compatible wallet
- 💰 **Minimum 5 ADA:** For transaction fees and UTxO requirements
- 🎨 **Unique NFT Token:** Required to identify your membership

[Complete Membership Guide](#membership-guide) →

### 2. 📝 Submitting Proposals

Once approved, submit proposals for funding: [Submit a New Proposal](/proposals/new)

[Complete Proposal Guide](#proposal-guide) →


## 👤 Membership Application Guide {#membership-guide}

### Step 1: Navigate to Application

Click `Become an Ambassador` in the navigation menu, or visit `/sign-up`

### Step 2: 🔗 Connect Your Wallet

- Click "Connect Wallet" and select your wallet provider
- Approve the connection request in your wallet
- Ensure you're on the correct network (Preview or Mainnet)

### Step 3: 🎨 Select Your NFT Token

The app will automatically detect eligible NFTs in your wallet. Choose the token for your membership.

> ⚠️ **Important:** This token will be locked in a smart contract during your membership.

### Step 4: 📋 Fill Out Application Form

All fields are stored on-chain:

**Required:**
- Full Name
- Display Name
- Email Address
- Bio

**Optional:**
- Country & City
- X (Twitter) Handle
- GitHub / Discord
- SPO ID / DRep ID

### Step 5: ✍️ Submit and Sign Transaction

- Review all your information carefully
- Click "Submit Application" and sign in your wallet
- Wait for blockchain confirmation (20-60 seconds)

### Step 6: 👀 Track Your Application

Navigate to `Submissions` (Member Tools):

- View status: Pending, Approved, or Rejected
- Monitor admin decision and signature progress

### Step 7: ✏️ Edit Your Application

- Go to Submissions and click your pending application
- Click "Edit Application", update fields, and save

> ⚠️ **Important:** Editing is disabled once admin signing starts.


## 📝 Proposal Submission Guide {#proposal-guide}

**For approved members only.**

### Step 1: Navigate to New Proposal

Click `Proposals` → `New Proposal`, or visit `/proposals/new`

### Step 2: 📋 Fill Out Proposal Details

- **Title:** Clear, descriptive title
- **Description:** Detailed explanation using the rich text editor
- **Funding Amount:** ADA requested (in lovelace)
- **Receiver Address:** Cardano wallet address for funds

### Step 3: 👀 Track and Edit Your Proposal

View proposals in `Submissions` (Member Tools):

- Click a proposal to view details and status
- Edit title and description using "Edit Proposal" button

> ⚠️ **Note:** Editing is disabled once admin signing starts.


## 🔄 Syncing Data

### Global Refresh Button

The app includes a **global refresh button** (located in the top navigation bar) to manually sync the latest data from the blockchain.

**When to Use:**

- After submitting a transaction and waiting for confirmation
- When checking for updates to your application or proposal status
- If signature counts don't appear up-to-date
- After other admins have signed decisions

**How It Works:**

- Click the refresh icon in the navigation bar
- The app will fetch the latest data from the blockchain
- All pages will update with current information
- Status changes and signature counts will reflect the latest state

> 💡 **Tip:** If you don't see expected updates after a transaction, wait 1-2 minutes for blockchain confirmation, then use the refresh button.


## ⚠️ Important Notes

- 🔒 All transactions are on-chain and **cannot be reversed**
- ✏️ Editing is disabled once admin signing begins
- 🔐 Never share your seed phrase with anyone
- 💸 Transaction fees vary based on network conditions
